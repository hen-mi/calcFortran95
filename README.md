
# Description
Basic console calculator made with Fortran (95).

Functions: 
- Sum;
- Subtraction;
- Multiplication;
- Division;
- Square numbers;
- Cube numbers.

Note: All values are real numbers, meaning it accepts either rational and/or irrational numbers.
